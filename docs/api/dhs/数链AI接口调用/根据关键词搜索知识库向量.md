**参数说明**

| 参数值     | 内容说明                             |
|---------|----------------------------------|
| code    | 200：表示成功响应                       |
| msg     | 成功响应信息                           |
| data    | 成功响应返回向量库中存储的内容，<br/>以列表形式存储多组信息 |
| name    | 文本对应的id                          |
| title   | 文本对应的问题或题目                       |
| id      | 表示向量id                           |
| content | 表示向量内容                           |
